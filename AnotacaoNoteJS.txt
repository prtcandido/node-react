Instalação Yarn

npm install --global yarn

yarn --version

Criar projeto NodeJS

yarn init -y

Instalando express

yarn add express

Instalando nodemon para não ter que restartar a aplicação a cada mudança. Parâmetro -D para apenas ambiente de desenvolvimento.

yarn add nodemon -D

Instalar typescript

yarn add typescript -D 

Instalar types do express

yarn add @types/express -D

Iniciar typescript no projeto

yarn tsc --init

Pacote necessário para criar script para execução da aplicação com typescript

yarn add ts-node-dev -D

Instalar tratamento de erros

yarn add express-async-errors
